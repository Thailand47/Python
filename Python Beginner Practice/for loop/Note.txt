การลูปผ่านตัวเลขตั้งแต่ x ถึง x และหาผลรวมของตัวเลขที่หารลงตัวด้วย x

โค้ดนี้ใช้ลูปผ่านตัวเลขจาก  x ถึง x และคัดกรองเฉพาะตัวเลขที่หารด้วย x ลงตัว (หรือเป็นเศษ x เมื่อหารด้วย x) แล้วเพิ่มผลลัพธ์เหล่านั้นเข้ามาในตัวแปร plus_total"

ใช้ if ทุกกรณี เมื่อเงื่อนไขแต่ละข้ออาจจะเกิดขึ้นพร้อมกัน และต้องการให้โปรแกรมตรวจสอบทุกเงื่อนไข
ใช้ elif เมื่อเราต้องการให้โปรแกรมเลือกเงื่อนไขใดเงื่อนไขหนึ่งเท่านั้น ถ้าเงื่อนไขแรกเป็นจริงแล้วจะข้ามเงื่อนไขที่เหลือ
ใช้ else เมื่อเงื่อนไขทั้งหมดไม่เป็นจริง (ไม่มีเงื่อนไขใดตรง) โปรแกรมจะทำงานในส่วนของ else แทน
การใช้ else จะเกิดขึ้นเมื่อไม่มีเงื่อนไขใน if หรือ elif เป็นจริงครับ
